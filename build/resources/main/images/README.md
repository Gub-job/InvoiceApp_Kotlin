# Logo Perusahaan

Letakkan file logo perusahaan dengan nama `logo.png` di folder ini.

Format yang didukung:
- PNG (Recommended)
- JPG/JPEG

Ukuran yang disarankan: 200x200 pixels atau lebih kecil untuk performa optimal.

Jika file logo.png tidak ditemukan, akan ditampilkan placeholder "[LOGO]" di Excel.
